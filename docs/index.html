Pine

Minimal native macOS code editor. Built with SwiftUI + AppKit, zero dependencies.

View on GitHub Download

Features

📁

File Tree Sidebar

Browse your project with a native file tree. Open folders, navigate files, see everything at a glance.

✎

Syntax Highlighting

Built-in grammar-based highlighting for popular languages. Extensible via simple JSON grammar files.

💻

Built-in Terminal

Integrated terminal powered by a real PTY. Run commands without leaving the editor.

🌀

Liquid Glass UI

Designed for macOS 26 Tahoe with native Liquid Glass materials and translucency effects.

📋

Native Tabs

Uses macOS native window tabs for a familiar, system-integrated experience.

🔌

Git Integration

See git status in the file tree, diff markers in the gutter, and switch branches easily.

"Pine is a pine tree in a forest of editors — quiet, reliable, native."

Pine doesn't try to be VS Code or Xcode. Open, write code, close. Each scale of the pine cone icon is a line of code.
