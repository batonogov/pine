Pine

Open a repo, edit code, run commands, check git, move on. Native macOS app. Zero dependencies.

Download View on GitHub

Features

📁

File Tree Sidebar

Browse a repo fast with a clean native sidebar, folders-first sorting, and no hidden-file clutter.

✎

Syntax Highlighting

Grammar-based highlighting for Swift, TypeScript, JavaScript, Python, Go, Rust, Shell, HTML, CSS, JSON, YAML, Markdown, and Dockerfile.

💻

Real PTY Terminal

Integrated terminal powered by a real PTY, with multiple tabs, so you can run commands without leaving the editor.

🌀

Liquid Glass UI

Designed for macOS 26 Tahoe with native Liquid Glass materials, translucency, and system behavior.

📋

Native Tabs

Uses real macOS window tabs instead of custom in-app chrome for a familiar, system-integrated workflow.

🔌

Git Integration

See git status in the file tree, diff markers in the gutter, branch switching, and a compact status summary.

"Less editor. More shipping."

Pine is for developers who want a native Mac editor with terminal and git context, without Electron overhead or plugin sprawl.
